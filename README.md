# ReuseNation

An app that serves the community the purpose for reusing the old products rather than converting it to waste just for recycling.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

* Android studio with proper plugins like kotlin and java jdk installed.
* An emulator with minSDK of 20 to run the app.

### Installing

A step by step series of examples that tell you how to get a development env running

```
git clone https://github.com/kirtikjangale/SeAndroidFrontend.git
```

Build the app by running it in android studio and test it on emulator or your personal phone.

## Deployment

Install the apk file by building it and then run on your devices.

## Features of App

* The app allows users to register themselves on app and then login to view the all functionalities of app.
* Then user can view the profile page and update the info if needed and upload a profile picture.

* The users can sell the item which has three categories like Electronics, textbooks, notebooks.
* The users can see the on-sell items by them in mylistings and remove the items when they are sold.
* The home page shows all the items on sale with two available filters, category{electronics, textbook and notes} and location by putting pincode.

* The user can see the items in their loction and add the product to wishlist and see the wishlisted items in wishlist section of profile.
* The indivdual product can be viewed by clicking on the card and all details regarding specifications, snapshots, thumbnail will be available.
